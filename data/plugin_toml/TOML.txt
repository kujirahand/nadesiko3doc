▲TOMLとは

TOML(Tom’s Obvious, Minimal Language)は設定ファイルを書くためのフォーマットです。「人間にとって読みやすく、機械にとって扱いやすい」ことを強く意識して作られたものです。

{{{#nako3(rows=10)
# TOML文字列をパースする方法
データ=「
# -----------------------
[setting]
foo = 3
bar = 5
# -----------------------
」
データをTOML取得して、Ｆに代入。
??F
}}}

JSONデータをTOML文字列として出力する例：

{{{#nako3(rows=10)
# JSONデータをTOMLに変換して出力する
データ={"setting": { "foo": 3, "bar": 5 } }
データをTOML変換して表示。
}}}
