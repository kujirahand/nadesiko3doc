■ドローンを制御しよう

なでしこv3をPCにインストールすると、なでしこでドローン(Telloシリーズ)を操作できます。離陸・着陸・移動・SDKのコマンド実行などをサポートしています。

●ドローン(Tello)を入手

・Tello SDK 1.3に対応したドローン(1万円前後)を入手します。

●nadesiko3-droneプラグインを入手

[[nadesiko3-droneプラグイン:https://github.com/kujirahand/nadesiko3-drone]]をインストールします。Node.jsがインストールされたPCであれば、コマンドラインから以下のようにして安定版をインストールできます。

{{{
$ cd (なでしこv3のフォルダ)
$ npm install nadesiko3-drone
}}}

●なでしことドローンの接続

Telloの電源を入れます。すると「Tello-***」というアクセスポイント(SSID)が見つかるので、PCのWi-FiをTelloに接続します。

以上で準備完了です。以下のプログラムを実行すると、Telloのバッテリを確認して表示します。

{{{
#
# ドローンを上下するだけのプログラム
#
!「nadesiko3-drone.js」を取り込む

「✈接続を開始しています」と表示。
ドローン接続
「接続しました。自動操縦はじめます」と表示。
ドローン離陸
「離陸しました」と表示
1秒待つ。
「上に移動」と表示
「上」に50だけドローン移動
1秒待つ
「下に移動」と表示
「下」に50だけドローン移動
1秒待つ
「離陸します」と表示
ドローン着陸
「自動操縦おわります」と表示。
}}}
