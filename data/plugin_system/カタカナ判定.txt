●『[[カタカナ判定:plugin_system/カタカナ判定]]』の詳しい解説

文字列Sの1文字目がカタカナか判定。
半角片仮名にはヒットしません。
{{{#nako3
「アイウエオ」をカタカナか判定して表示。　# true
「アいうえお」をカタカナか判定して表示。　# true
「あイウエオ」をカタカナか判定して表示。　# false
「ｱｲｳｴｵ」をカタカナか判定して表示。　　　　# false
}}}
