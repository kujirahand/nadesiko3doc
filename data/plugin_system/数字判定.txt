●『[[数字判定:plugin_system/数字判定]]』の詳しい解説
文字列Sの1文字目が数字か判定。
全角半角共にヒットします。
{{{#nako3
「12345」を数字か判定して表示。　　　　# true
「１２３４５」を数字か判定して表示。　# true
「12月24日」を数字か判定して表示。　　# true
}}}
